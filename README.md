## Install Typescript
```bash
npm --version

npm install typescript nodemon -D
#atau
sudo npm install -g typescript

tsc --version

```

Untuk dokumentasi di sini ([Praktek Typescript](./Script.md))

## Menjalankan Typescript script
Buka terminal 1
```bash
npm run tsc
```
Buka terminal 2
```bash
npm run dev-1
#s/d
npm run dev-9
```
